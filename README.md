- Express

## Gettin Started

---

### Server

Commands must be run from the `server` directory:

#### Installation

```bash
npm install
```

#### Run

```bash
npm run start:dev
```

#### Tests

Just linting:

```bash
npm run lint
```

Run linting, and fix any errors:

```bash
npm run lint:fix
```

Run Jest tests:

```
npm run jest
```

Run Jest tests, and watch for changes:

```bash
npm run jest:watch
```

Run all tests:

```bash
npm run test
```
